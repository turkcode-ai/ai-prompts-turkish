# Sosyal Medya Promptları

## 1. Instagram Post Serisi

```
[Konu] hakkında 7 günlük Instagram içerik serisi oluştur.

Marka: [Marka adı]
Sektör: [Sektör]
Hedef kitle: [Kim]
Ses tonu: [Samimi/Profesyonel/Eğlenceli]

Her gün için:
- Görsel fikri (detaylı açıklama)
- Caption (emoji ile)
- Call-to-action
- 5-10 hashtag
- En iyi paylaşım saati

Pazartesi: Motivasyon/Haftalık hedef
Salı: Eğitici içerik
Çarşamba: Behind the scenes
Perşembe: Kullanıcı içeriği/Testimonial
Cuma: Fun/Eğlenceli içerik
Cumartesi: Ürün/Hizmet tanıtımı
Pazar: Haftalık özet/Engagement post
```

## 2. LinkedIn Düşünce Liderliği Postu

```
[Konu] hakkında viral olabilecek LinkedIn postu yaz.

Hedef: [Farkındalık/Tartışma/Leads]
Perspektif: [Sektör içgörüsü/Kişisel deneyim/Trend analizi]

Post yapısı:
1. Hook - İlk cümle dikkat çekici (soru/istatistik/bold iddia)
2. Problem - Herkesin bildiği ama konuşmadığı
3. İçgörü - Beklenmedik bakış açısı
4. Hikaye - Kişisel deneyim (kısa)
5. Actionable takeaway - Somut öneriler
6. CTA - Yorum/Paylaşım teşviki

Kısa paragraflar, kolay okunabilir format.
```

## 3. Twitter/X Thread

```
[Konu] hakkında viral Twitter thread yaz.

Tarz: [Eğitici/Hikaye/Liste]
Hedef: [Follower/Engagement/Link tıklama]

Thread yapısı (10 tweet):
Tweet 1: Hook - Thread'i okumak için neden durmalı
Tweet 2-8: Ana içerik (her tweet tek fikir)
Tweet 9: Özet/Key takeaways
Tweet 10: CTA + RT talebi

Her tweet:
- Max 280 karakter
- Emoji kullanımı
- Kendi başına anlamlı olsun
- Bir sonrakini merak ettirsin
```

## 4. YouTube Video Script

```
[Konu] hakkında [X] dakikalık YouTube videosu için script yaz.

Kanal türü: [Eğitim/Vlog/Review]
Hedef kitle: [Kim]
Video amacı: [Eğitmek/Eğlendirmek/Satış]

Script:
1. Hook (0-10 saniye) - İzleyiciyi yakala
2. Intro (10-30 saniye) - Konu tanıtımı + değer vaadi
3. Ana içerik (bölümlere ayrılmış)
4. B-roll önerileri (parantez içinde)
5. CTA'lar (video ortası + sonu)
6. Outro - Özet + sonraki video teaser

Ekstra:
- Başlık önerileri (3 adet)
- Thumbnail fikri
- Açıklama ve etiketler
```

## 5. TikTok/Reels Script

```
[Konu] hakkında viral TikTok/Reels script'i yaz.

Format: [Tutorial/POV/Story time/Trend]
Süre: [15/30/60 saniye]
Trend müziği: [Varsa belirt]

Script:
1. Hook (ilk 1-2 saniye) - ÇOOK önemli
2. Setup (5-10 saniye)
3. Content/Value (ana kısım)
4. Punch/Payoff (son saniyeler)
5. CTA (takip/kaydet/paylaş)

Notlar:
- Text overlay önerileri
- Geçiş efektleri
- Ses kullanımı
```
