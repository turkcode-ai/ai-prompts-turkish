## 💼 İş & Kariyer

### CV Hazırlama

```
Bir kariyer danışmanı olarak hareket et. [X yıl] deneyimli bir [meslek] için modern ve ATS uyumlu bir CV hazırla.

Deneyimlerim:
- [Şirket 1]: [Pozisyon], [Tarih aralığı]
- [Şirket 2]: [Pozisyon], [Tarih aralığı]

Yeteneklerim: [Yetenek listesi]

CV şu özelliklerde olsun:
1. Profesyonel özet (3-4 cümle)
2. Deneyimler (başarı odaklı, rakamlarla destekli)
3. Yetenekler (kategorize edilmiş)
4. Eğitim

Modern format kullan, gereksiz detaylardan kaçın.
```

### Mülakat Hazırlık

```
[Şirket] şirketinde [Pozisyon] pozisyonu için mülakat koçum ol.

Şirket hakkında: [Kısa açıklama veya sektör]
Deneyimim: [Yıl] yıl [Alan]

Şunları yap:
1. Bu pozisyon için en sık sorulan 10 mülakat sorusunu listele
2. Her soru için STAR metoduyla örnek cevap ver
3. Bu şirkete özel sorulabilecek 5 soru öner
4. Mülakatın sonunda sormam gereken 3 akıllı soru öner
5. Kaçınmam gereken 5 kritik hatayı listele
```

### İş Başvuru E-postası

```
[Şirket] şirketinin [Pozisyon] ilanına başvuru için profesyonel bir e-posta yaz.

Benim hakkımda:
- [X] yıl [sektör] deneyimi
- [Öne çıkan yetenek 1]
- [Öne çıkan yetenek 2]
- [İlgili başarı]

E-posta şöyle olsun:
- Dikkat çekici açılış
- Neden bu pozisyon ve şirket
- Değer önerim (ne katabilirim)
- Harekete geçirici kapanış
- Profesyonel ama sıcak ton
```

### LinkedIn Profil Optimizasyonu

```
LinkedIn profilimi optimize et. [Sektör] sektöründe [hedef pozisyon] arıyorum.

Mevcut başlığım: [Başlık]
Deneyimim: [Özet]

Şunları oluştur:
1. Dikkat çekici başlık (120 karakter)
2. Hakkımda bölümü (2000 karakter, anahtar kelime optimizasyonlu)
3. Deneyim açıklamaları (başarı odaklı)
4. Yetenekler listesi (önem sırasına göre 20 yetenek)
5. İşe alımcıların dikkatini çekecek öne çıkan bölüm önerileri
```

### Maaş Pazarlığı

```
Maaş pazarlığı koçum ol. [Pozisyon] için teklif aldım.

Teklif edilen: [Miktar]
Piyasa ortalaması: [Araştırman varsa]
Benim beklentim: [Miktar]

Şunları hazırla:
1. Pazarlık için açılış konuşması
2. Değerimi gösteren 5 argüman
3. Karşı tekliflere nasıl cevap vereceğim
4. Maaş dışı pazarlık edilebilecekler (bonus, tatil, uzaktan çalışma)
5. Anlaşma sağlanamazsa B planı
```

## 💻 Yazılım & Kod

### Kod Review

```
Kıdemli bir yazılım mühendisi olarak bu kodu incele:

```[programlama dili]
[Kod buraya]
```

Şu açılardan değerlendir:
1. **Performans**: Optimizasyon önerileri
2. **Güvenlik**: Potansiyel açıklar
3. **Okunabilirlik**: Clean code prensipleri
4. **Best Practices**: Sektör standartları
5. **Refactoring**: Somut iyileştirme önerileri

Her öneri için "önce/sonra" kod örneği ver.
```

### Bug Çözme

```
Debugging uzmanı olarak bu hatayı çözmeme yardım et:

**Hata mesajı:**
```
[Hata mesajını yapıştır]
```

**İlgili kod:**
```[dil]
[Kodu yapıştır]
```

**Beklenen davranış:** [Ne olması gerekiyordu]
**Gerçekleşen:** [Ne oluyor]

Şunları yap:
1. Hatanın kök nedenini açıkla
2. Adım adım çözüm sun
3. Bu hatanın tekrar oluşmaması için öneri ver
4. Benzer hatalar için genel debugging stratejisi öner
```

### API Tasarımı

```
[Proje türü] için RESTful API tasarla.

Gereksinimler:
- [Özellik 1]
- [Özellik 2]
- [Özellik 3]

Şunları oluştur:
1. Endpoint listesi (HTTP metodları ile)
2. Request/Response örnekleri (JSON)
3. Hata kodları ve mesajları
4. Authentication stratejisi
5. Rate limiting önerisi
6. OpenAPI/Swagger dokümantasyonu
```

### Veritabanı Tasarımı

```
[Proje] için veritabanı şeması tasarla.

Gereksinimler:
- [Özellik/Entity 1]
- [Özellik/Entity 2]
- [Özellik/Entity 3]

Şunları sağla:
1. ER diyagramı (metin formatında)
2. Tablo yapıları (sütunlar, tipler, kısıtlamalar)
3. İlişkiler (1-1, 1-N, N-N)
4. İndeks önerileri
5. Örnek SQL sorguları (CRUD)
6. Normalizasyon seviyesi açıklaması
```

### Git Commit Mesajı

```
Şu değişiklikler için conventional commit mesajları yaz:

Değişiklikler:
[Değişiklik listesi]

Her commit için:
- Conventional commit formatı (feat/fix/docs/style/refactor/test/chore)
- Kısa başlık (50 karakter max)
- Detaylı açıklama (gerekirse)
- Breaking change varsa belirt
```

### Kod Açıklama

```
Bu kodu detaylı açıkla:

```[dil]
[Kod buraya]
```

Şu şekilde açıkla:
1. Genel amaç (1-2 cümle)
2. Satır satır açıklama
3. Kullanılan algoritmalar/pattern'ler
4. Zaman ve alan karmaşıklığı (Big O)
5. Potansiyel iyileştirmeler
6. Benzer alternatiflerin karşılaştırması
```

## ✍️ İçerik Üretimi

### Blog Yazısı

```
[Konu] hakkında SEO uyumlu blog yazısı yaz.

Hedef kitle: [Kim]
Anahtar kelime: [Ana anahtar kelime]
Ton: [Profesyonel/Samimi/Eğitici]
Uzunluk: [Kelime sayısı]

Yapı:
1. Dikkat çekici başlık (H1) - 60 karakter
2. Meta açıklama - 155 karakter
3. Giriş - Hook ile başla
4. Ana bölümler (H2'ler ile)
5. Pratik örnekler/ipuçları
6. Sonuç - CTA ile bitir

SEO gereksinimleri:
- Anahtar kelime yoğunluğu %2-3
- İç ve dış link önerileri
- Görsel önerileri (alt text ile)
```

### Sosyal Medya Paketi

```
[Konu/Ürün/Hizmet] için sosyal medya içerik paketi oluştur.

Platform: [Instagram/Twitter/LinkedIn/Hepsi]
Hedef: [Farkındalık/Satış/Engagement]

Her platform için:
1. 5 farklı post metni
2. Hashtag önerileri
3. En iyi paylaşım saatleri
4. Görsel/video fikri
5. CTA önerileri

Bonus: 1 haftalık içerik takvimi
```

### E-posta Serisi

```
[Ürün/Hizmet] için e-mail pazarlama serisi yaz.

Hedef: [Satış/Onboarding/Nurturing]
Seri uzunluğu: [X] e-mail

Her e-mail için:
1. Konu satırı (A/B test için 2 versiyon)
2. Ön izleme metni
3. E-mail gövdesi
4. CTA butonu metni
5. Gönderim zamanlaması

Seri akışı:
- Email 1: [Amaç]
- Email 2: [Amaç]
- Email 3: [Amaç]
...
```

### Video Script

```
[Konu] hakkında [X] dakikalık YouTube videosu için script yaz.

Kanal türü: [Eğitim/Eğlence/Vlog]
Hedef kitle: [Kim]

Script şablonu:
1. Hook (ilk 5 saniye) - İzleyiciyi yakala
2. Intro (10-15 saniye) - Konuyu tanıt
3. Ana içerik - Bölümlere ayrılmış
4. B-roll önerileri
5. Outro - Abone ol + sonraki video teaser

Ekstra:
- Thumbnail fikri
- Başlık önerileri (3 adet)
- Açıklama ve etiketler
```

## 📚 Eğitim & Öğrenme

### Konu Öğrenme

```
[Konu] konusunu bana öğret.

Seviyem: [Başlangıç/Orta/İleri]
Öğrenme stilim: [Görsel/İşitsel/Pratik]
Günlük ayırabileceğim süre: [X saat]

Şunları hazırla:
1. Konunun 30 saniyelik özeti
2. Öğrenme yol haritası (haftalık plan)
3. Temel kavramlar listesi
4. Pratik alıştırmalar
5. Önerilen kaynaklar (ücretsiz)
6. Yaygın hatalar ve çözümleri
7. İlerlemeyi ölçme kriterleri
```

### Sınav Hazırlık

```
[Sınav adı] sınavına hazırlanmama yardım et.

Sınav tarihi: [Tarih]
Mevcut seviyem: [Açıklama]
Hedef puanım: [Puan]

Şunları oluştur:
1. Konu bazlı çalışma planı
2. Her konu için özet notlar
3. 20 çıkmış/çıkabilecek soru
4. Soru çözme stratejileri
5. Son hafta tekrar planı
6. Sınav günü taktikleri
```

### Dil Öğrenme

```
[Dil] öğrenmek istiyorum.

Mevcut seviyem: [A1-C2 arası]
Hedefim: [Konuşma/Yazma/İş/Seyahat]
Günlük sürem: [X dakika]

Şunları hazırla:
1. 30 günlük öğrenme planı
2. Günlük kelime listesi (10 kelime)
3. Temel gramer kuralları
4. Pratik diyalog örnekleri
5. Dinleme/izleme önerileri
6. Konuşma partneri bulma önerileri
```

## 📈 Pazarlama & Sosyal Medya

### Pazarlama Stratejisi

```

İşletme türü: [B2B/B2C]
Hedef kitle: [Demografik bilgiler]
Bütçe: [Aylık bütçe]
Hedef: [Satış/Lead/Farkındalık]

Strateji şunları içersin:
1. SWOT analizi
2. Kanal stratejisi (hangi platformlar, neden)
3. İçerik stratejisi (türler, sıklık)
4. Reklam stratejisi (bütçe dağılımı)
5. KPI'lar ve ölçüm yöntemleri
6. 3 aylık aksiyon planı
7. Rakip analizi
```

Hedef kitle: [Kim]
Ürün faydaları: [Liste]
Kampanya hedefi: [Satış/Trafik/Lead]

Şunları oluştur:
1. 5 farklı başlık (karakter limitlerine uygun)
2. 3 farklı açıklama metni
3. CTA seçenekleri
4. A/B test önerileri
5. Negatif anahtar kelimeler
6. Hedefleme önerileri
```

### Influencer İşbirliği

```
[Marka] için influencer pazarlama stratejisi oluştur.

Ürün/Hizmet: [Açıklama]
Bütçe: [Miktar]
Hedef platform: [Instagram/TikTok/YouTube]

Şunları hazırla:
1. İdeal influencer profili
2. Influencer araştırma kriterleri
3. İşbirliği teklifi e-mail şablonu
4. Brief şablonu
5. Performans metrikleri
6. Sözleşme maddesi önerileri

## 🎯 Mega Promptlar

### Rol Bazlı Asistan

```
Sen şu andan itibaren [ROL] olarak davranacaksın.

Rol özelliklerin:
- [Uzmanlık alanı]
- [Deneyim seviyesi]
- [Çalışma stili]

İletişim stilin:
- [Ton: Resmi/Samimi/Teknik]
- [Detay seviyesi: Özet/Detaylı]
- [Örneklerle mi açıkla]

Her yanıtında:
1. [Yapı kuralı 1]
2. [Yapı kuralı 2]
3. [Yapı kuralı 3]

Bu rolü "RESET" deyene kadar sürdür.
```

### Sokratik Öğretmen

```
Sokratik yöntemle öğreten bir eğitimci ol.

Konu: [Öğrenmek istediğim konu]
Seviyem: [Başlangıç/Orta/İleri]

Kurallar:
1. Direkt cevap VERME
2. Düşündürücü sorular sor
3. Yanlışlarımı sorularla fark ettir
4. Doğru yöne yönlendir
5. Küçük ipuçları ver
6. Kavrayıp kavramadığımı test et

"CEVAP VER" dersem normal cevap ver.
```

### Çok Perspektifli Analiz

```
[Konu/Karar] konusunu farklı perspektiflerden analiz et.

Perspektifler:
1. 🎯 Stratejist: Uzun vadeli etkileri
2. 😈 Şeytanın Avukatı: Karşı argümanlar
3. 🔢 Analist: Veriler ve rakamlar
4. 🎨 Yaratıcı: Alternatif çözümler
5. 👤 Son Kullanıcı: Müşteri gözü
6. ⚖️ Risk Yöneticisi: Riskler ve önlemler

Sonuç olarak:
- Konsensüs noktaları
- Çatışan görüşler
- Önerilen aksiyon
```
